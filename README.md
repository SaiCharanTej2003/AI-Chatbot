This is a lightweight Virtual Chatbot that uses the Groq API (OpenAI-compatible endpoint) for model inference.

Features -
1.Express backend that forwards chat requests to Groq.
2.React + Vite frontend chat UI.

Configure via backend/.env.
Edit backend/.env with your Groq API key and model.

Backend:
cd backend
npm install
npm start

Frontend:
cd frontend
npm install
npm run dev

Open the Vite URL (usually http://localhost:5173) and chat.
